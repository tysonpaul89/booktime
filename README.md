# booktime

A demo Django app for learning. Referenced from the book **Practical Django 2 and Channels 2**

### Requirements:
- Python 3+
- Django 2

### Installation:
`git clone https://github.com/tysonpaul89/booktime.git`
`cd booktime`
`pipenv --three`
`pipenv shell`
`pip install django`
`./manage.py runserver`
